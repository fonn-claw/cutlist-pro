**BoardPresets.tsx** -- Preset dropdown that calls onSelect with preset data:
- Props: `{ onSelect: (lengthMm: number, widthMm: number) => void }`
- `'use client'` directive
- Renders a `<select>` element with a disabled "Select a preset..." placeholder option
- Maps BOARD_PRESETS to `<option>` elements with preset.name as display text

- Resets select to placeholder after selection
- Styled with: `w-full p-2 rounded bg-surface-alt border border-border text-text-primary text-sm`

**BoardForm.tsx** -- Add form with length, width, quantity inputs:
- Props: `{ onAdd: (board: Omit<Board, 'id'>) => void, prefilledLength?: number, prefilledWidth?: number }`
- `'use client'` directive
- Three controlled inputs: length (string state), width (string state), quantity (string state, default "1")
- Number inputs with `type="number"`, `min="0.01"` for dimensions, `min="1"` and `step="1"` for quantity
- `step="any"` on dimension inputs for maximum flexibility
- Prefilled values: use useEffect to update length/width input strings when prefilledLength/prefilledWidth props change. Convert mm to display units via `toDisplay(prefilledLength, units)` before setting input string.
- On submit: validate all fields (length > 0, width > 0, quantity >= 1 integer). If invalid, do nothing. If valid, convert length/width from display units to mm via `toInternal(parseFloat(lengthInput), units)`, call onAdd with the board object, then clear all fields back to defaults.
- Validation function: `parseFloat(value)` must be > 0 for dimensions, `parseInt(value)` must be >= 1 for quantity
- Add button: disabled when form is invalid. Styled with `bg-accent text-surface font-medium px-4 py-2 rounded hover:bg-amber-600 disabled:opacity-50 disabled:cursor-not-allowed`
- Layout: length and width inputs side by side (flex row, gap-2), quantity below, Add button full width below that
- Input styling: `w-full p-2 rounded bg-surface-alt border border-border text-text-primary text-sm`
- Labels: `text-xs text-text-secondary` above each input

**BoardEntry.tsx** -- Single board display/edit card:
- Props: `{ board: Board, isEditing: boolean, onStartEdit: () => void, onSave: (updates: Partial<Omit<Board, 'id'>>) => void, onCancelEdit: () => void, onRemove: () => void }`
- `'use client'` directive
- Uses `useUnits()` for unit-aware display and editing
- **Display mode** (isEditing=false): Show formatted dimensions via `formatDimension(board.dimensions.length, units)` x `formatDimension(board.dimensions.width, units)`, quantity as "qty: {n}". Click anywhere on the card calls onStartEdit. Delete button (X icon, `text-text-secondary hover:text-red-500`) on the right side calls onRemove.
- **Edit mode** (isEditing=true): Show three inputs (length, width, quantity) pre-filled with current values converted to display units via `toDisplay()`. Save button and Cancel button. On save: convert inputs back to mm via `toInternal()`, call onSave with new dimensions and quantity. On cancel: call onCancelEdit.
- Card styling: `p-3 rounded border border-border bg-surface-alt mb-2 cursor-pointer hover:border-accent` in display mode. Remove cursor-pointer and hover in edit mode.

**BoardList.tsx** -- Board list container managing edit state:

- Maps boards to BoardEntry components with key={board.id}
- Passes isEditing={editingId === board.id}
- onStartEdit sets editingId to board.id
- onSave calls onUpdate then sets editingId to null
- onCancelEdit sets editingId to null
- onRemove calls onRemove then sets editingId to null if removing the editing board

Rewrite src/app/page.tsx to be a client component that owns board state:

```typescript
'use client';

import { useState, useCallback } from 'react';
import type { Board } from '@/lib/types';
import { addBoard, updateBoard, removeBoard } from '@/lib/board-operations';
import { Header } from '@/components/layout/Header';
import { Sidebar } from '@/components/layout/Sidebar';
import { MainArea } from '@/components/layout/MainArea';
import { BoardPresets } from '@/components/boards/BoardPresets';
import { BoardForm } from '@/components/boards/BoardForm';
import { BoardList } from '@/components/boards/BoardList';

export default function Home() {
  const [boards, setBoards] = useState<Board[]>([]);
  const [prefilledLength, setPrefilledLength] = useState<number | undefined>();
  const [prefilledWidth, setPrefilledWidth] = useState<number | undefined>();

  const handleAdd = useCallback((board: Omit<Board, 'id'>) => {
    setBoards(prev => addBoard(prev, board));
    setPrefilledLength(undefined);
    setPrefilledWidth(undefined);
  }, []);

  const handleUpdate = useCallback((id: string, updates: Partial<Omit<Board, 'id'>>) => {
    setBoards(prev => updateBoard(prev, id, updates));
  }, []);

  const handleRemove = useCallback((id: string) => {
    setBoards(prev => removeBoard(prev, id));
  }, []);

  const handlePresetSelect = useCallback((lengthMm: number, widthMm: number) => {
    setPrefilledLength(lengthMm);
    setPrefilledWidth(widthMm);
  }, []);

  return (
    <div className="h-screen flex flex-col">
      <Header />
      <div className="flex-1 flex flex-col md:flex-row overflow-hidden">
        <Sidebar>
          <BoardPresets onSelect={handlePresetSelect} />
          <BoardForm
            onAdd={handleAdd}
            prefilledLength={prefilledLength}
            prefilledWidth={prefilledWidth}
          />
          <BoardList
            boards={boards}
            onUpdate={handleUpdate}
            onRemove={handleRemove}
          />
        </Sidebar>
        <MainArea />
      </div>
    </div>
  );
}
```

This converts page.tsx from a server component to a client component (adding 'use client'). Board state is lifted here so future phases (optimization, visualization) can access the boards array. The preset-to-form flow works by: preset dropdown calls handlePresetSelect which sets prefilled mm values, BoardForm receives these as props and converts to display units for the input fields.
