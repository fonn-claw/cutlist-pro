    Replace the stub in src/lib/optimizer.ts with the full implementation. The file should contain these internal types and functions:

    **Internal type (not exported):**
    ```typescript
    interface FreeRectangle {
      x: number;
      y: number;
      width: number;
      height: number;
    }
    ```

    **Internal helper: effectiveDimensions(piece, kerf)**
    Returns `{ width: piece.dimensions.length + kerf, height: piece.dimensions.width + kerf }`.
    Note: `dimensions.length` maps to placed `width`, `dimensions.width` maps to placed `height`.
    Adding full kerf to each dimension (standard approach -- slightly overestimates at board edges, but produces safe layouts).

    **Internal helper: placePieceInRect(freeRects, pieceWidth, pieceHeight, canRotate)**
    Implements Best Short Side Fit (BSSF):
    - Iterate all free rectangles
    - For each rect, check if piece fits in normal orientation (pieceWidth <= rect.width AND pieceHeight <= rect.height)
    - Score = min(rect.width - pieceWidth, rect.height - pieceHeight) -- the shorter leftover side
    - If canRotate, also check rotated orientation (pieceHeight <= rect.width AND pieceWidth <= rect.height), score similarly
    - Return the rect with lowest score (best fit), along with whether rotated
    - Return null if no rect fits
    - Use epsilon = 0.01 for float comparisons (pieceWidth <= rect.width + epsilon)

    **Internal helper: splitFreeRect(rect, placedWidth, placedHeight)**
    Implements Short Axis Split:
    - After placing piece at (rect.x, rect.y) with size (placedWidth, placedHeight):
    - rightWidth = rect.width - placedWidth
    - bottomHeight = rect.height - placedHeight
    - If both > 0: compare rightWidth vs bottomHeight
      - If rightWidth < bottomHeight (right is shorter axis): vertical cut
        - Right remainder: {x: rect.x + placedWidth, y: rect.y, width: rightWidth, height: placedHeight}
        - Bottom remainder: {x: rect.x, y: rect.y + placedHeight, width: rect.width, height: bottomHeight}
      - Else: horizontal cut
        - Right remainder: {x: rect.x + placedWidth, y: rect.y, width: rightWidth, height: rect.height}
        - Bottom remainder: {x: rect.x, y: rect.y + placedHeight, width: placedWidth, height: bottomHeight}
    - Filter out any rectangles with width < 0.01 or height < 0.01

    **Main function: optimizeCutLayout(boards, pieces, settings)**
    1. Handle empty inputs: if boards.length === 0 or pieces.length === 0, return empty result with zeroed summary
    2. Expand board quantities: Board{id:'b1', qty:3} becomes 3 board instances [{boardId:'b1', instanceIndex:0}, ...]
    3. Expand piece quantities: CutPiece{id:'p1', qty:2} becomes 2 piece instances [{...piece, instanceIndex:0}, {... ,instanceIndex:1}]
    4. Sort expanded pieces by area descending: (dimensions.length * dimensions.width), largest first
    5. Initialize activeBoards: BoardLayout[] = [] (no boards pre-allocated)
    6. Initialize availableBoardStock from expanded boards (queue to allocate from)
    7. For each piece instance:
       a. Compute effective dims with kerf: effectiveDimensions(piece, settings.kerf)
       b. canRotate = !piece.grainDirection
       c. Try placePieceInRect on each active board's freeRectangles
       d. Pick the board with the best score (lowest BSSF score across all boards)
       e. If placed: remove the chosen freeRect, add splitFreeRect results, add PlacedPiece to board.pieces
          - PlacedPiece uses ORIGINAL dimensions (not kerf-expanded): if not rotated, width=dimensions.length, height=dimensions.width; if rotated, width=dimensions.width, height=dimensions.length
       f. If no active board fits: allocate next board from availableBoardStock
          - New board starts with freeRectangles: [{x:0, y:0, width: board.dimensions.length, height: board.dimensions.width}]
          - Retry placement on new board
       g. If no stock boards left: add to unplacedPieces
    8. For each active board, compute:
       - usedArea = sum of (piece.width * piece.height) for all placed pieces
       - wasteArea = (board.width * board.height) - usedArea
       - utilizationPercent = (usedArea / (board.width * board.height)) * 100
       - waste = convert remaining freeRectangles to WasteRegion[]
    9. Build summary:
       - totalBoards = activeBoards.length
       - totalPieces = total expanded piece instances
       - placedPieces = totalPieces - unplacedPieces.length
       - totalArea = sum of (board.width * board.height) for active boards
       - usedArea = sum across boards
       - wasteArea = totalArea - usedArea
       - wastePercentage = (wasteArea / totalArea) * 100 (or 0 if totalArea is 0)

    **Critical constraints:**
    - NEVER mutate input boards or pieces arrays. Clone/spread at start.
    - Epsilon 0.01mm for all float comparisons.
    - Filter zero-dimension free rects after every split.
