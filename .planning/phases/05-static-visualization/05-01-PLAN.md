Create 4 components in `src/components/visualization/`:

**PieceRect.tsx** — `'use client'` component receiving `{ piece: PlacedPiece; units: UnitSystem }` props.
- Renders an SVG `<g>` containing:
  - `<rect>` at `x={piece.x}` `y={piece.y}` `width={piece.width}` `height={piece.height}` with `fill={piece.color}` `fillOpacity={0.7}` `stroke={piece.color}` `strokeWidth={1}`
  - Cursor style `cursor="pointer"` on the rect for tooltip affordance
- Label visibility: Show label text only when `piece.width >= 40 && piece.height >= 20` (mm units in SVG coordinate space)
- Font size: `Math.min(piece.width / 8, piece.height / 3, 12)` capped at 12 SVG units
- Two `<text>` elements centered in the piece:
  - Line 1: `piece.label` at `y = piece.y + piece.height/2 - fontSize*0.3`
  - Line 2: `formatDimension(piece.width, units) + ' x ' + formatDimension(piece.height, units)` at `y = piece.y + piece.height/2 + fontSize*0.8`
  - Both: `textAnchor="middle"` `dominantBaseline="middle"` `x={piece.x + piece.width/2}`
  - Fill color from `getContrastTextColor(piece.color)`, second line at `fillOpacity={0.8}` with `fontSize * 0.75`
- Export `onMouseEnter`, `onMouseMove`, `onMouseLeave` prop passthrough on the outer `<g>` for tooltip wiring in Plan 02

**WasteRect.tsx** — Component receiving `{ region: WasteRegion; boardIndex: number }` props.
- Renders `<rect>` at region x,y,width,height with `fill={`url(#waste-hatch-${boardIndex})`}` `stroke="var(--border)"` `strokeWidth={0.5}`
- Does NOT define the pattern itself (pattern defined in BoardDiagram's `<defs>`)

**BoardDiagram.tsx** — `'use client'` component receiving `{ layout: BoardLayout; boardIndex: number; totalBoards: number }` props.
- Uses `useUnits()` to get current unit system
- Computes viewBox via `calculateViewBox(layout.width, layout.height)`
- Structure:
  - Outer `<div>` with board header: `<h3>` showing `Board {boardIndex + 1} of {totalBoards} — {formatDimension(layout.width, units)} x {formatDimension(layout.height, units)}`
  - Header also shows utilization: `({layout.utilizationPercent.toFixed(1)}% used)`
  - Container `<div className="overflow-hidden border border-border rounded-lg bg-surface relative">` (the `relative` is for tooltip positioning in Plan 02)
  - Inner `<div>` for zoom/pan transform (Plan 02 will add transform styles; for now just a plain wrapper)
  - `<svg viewBox={viewBox} className="w-full h-auto" style={{ maxHeight: '600px' }} preserveAspectRatio="xMidYMid meet">`
  - Inside SVG `<defs>`:
    ```
    <pattern id={`waste-hatch-${boardIndex}`} patternUnits="userSpaceOnUse"
      width={8} height={8} patternTransform="rotate(45)">
      <line x1={0} y1={0} x2={0} y2={8}
        className="stroke-text-secondary" strokeWidth={0.5} strokeOpacity={0.3} />
    </pattern>
    ```
    NOTE: Use inline style `stroke="var(--text-secondary)"` instead of className on SVG `<line>` since Tailwind stroke classes may not work on SVG pattern elements.
  - Board outline: `<rect x={0} y={0} width={layout.width} height={layout.height} fill="none" className="stroke-border" strokeWidth={1} />`
  - Map `layout.waste` to `<WasteRect>` components with `boardIndex` prop
  - Map `layout.pieces` to `<PieceRect>` components with `units` prop

**CuttingDiagramList.tsx** — `'use client'` component receiving `{ result: OptimizationResult }` props.

- Maps filtered boards to `<BoardDiagram>` with `key={`${layout.boardId}-${layout.instanceIndex}`}`
