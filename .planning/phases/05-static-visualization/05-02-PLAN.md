**Create `src/hooks/useZoomPan.ts`:**

Custom hook `useZoomPan(minZoom = 1, maxZoom = 4)` returning:
```typescript
interface ZoomPanState { zoom: number; panX: number; panY: number; }
// Returns: { zoom, panX, panY, containerRef, handlePointerDown, handlePointerMove, handlePointerUp, reset }
```

Implementation details:
- `useState<ZoomPanState>({ zoom: 1, panX: 0, panY: 0 })`
- `containerRef = useRef<HTMLDivElement>(null)`
- `isPanning = useRef(false)`, `lastPointer = useRef({ x: 0, y: 0 })`
- `handleWheel` (via useCallback): `e.preventDefault()`, compute `delta = e.deltaY > 0 ? 0.9 : 1.1`, clamp `prev.zoom * delta` between minZoom and maxZoom
- `handlePointerDown`: set `isPanning.current = true`, store `e.clientX/Y` in lastPointer, call `(e.target as HTMLElement).setPointerCapture(e.pointerId)`
- `handlePointerMove`: if not panning return; compute `dx = e.clientX - lastPointer.current.x`, `dy = e.clientY - lastPointer.current.y`; update lastPointer; setState `panX + dx/zoom, panY + dy/zoom`
- `handlePointerUp`: `isPanning.current = false`
- `reset`: setState `{ zoom: 1, panX: 0, panY: 0 }`
- `useEffect` attaching wheel listener with `{ passive: false }` to containerRef.current, cleanup on unmount
- IMPORTANT: Use `addEventListener('wheel', ...)` in useEffect, NOT React `onWheel` prop — React's synthetic event does not support `{ passive: false }` which is required for `e.preventDefault()` to work

**Create `src/hooks/useZoomPan.test.ts`:**

Test the pure computation logic by extracting or testing state transitions:

- Test that repeated zoom out from 1: stays at 1
