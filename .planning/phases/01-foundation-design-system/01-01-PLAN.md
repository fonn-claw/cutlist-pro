4. Create `src/lib/units.ts` implementing these exact exports (TDD — green phase):
   - `const MM_PER_INCH = 25.4`
   - `toInternal(value: number, units: UnitSystem): number` — converts display value to mm. Imperial multiplies by 25.4, metric returns as-is.
   - `toDisplay(valueMm: number, units: UnitSystem): number` — converts mm to display value. Imperial divides by 25.4, metric returns as-is.
   - `formatFraction(decimalInches: number, precision: number = 16): string` — converts decimal inches to fraction string. Algorithm:
     a. Split into wholeInches = Math.floor(decimalInches) and remainder = decimalInches - wholeInches
     b. Round remainder to nearest 1/precision: sixteenths = Math.round(remainder * precision)
     c. Handle edge: if sixteenths === precision, increment wholeInches and set sixteenths = 0
     d. If sixteenths === 0, return `${wholeInches}"`
     e. Simplify fraction with GCD: gcd(a, b) => b === 0 ? a : gcd(b, a % b)
     f. numerator = sixteenths / gcd(sixteenths, precision), denominator = precision / gcd(sixteenths, precision)
     g. If wholeInches > 0, return `${wholeInches}-${numerator}/${denominator}"`
     h. Otherwise return `${numerator}/${denominator}"`
   - `formatDimension(valueMm: number, units: UnitSystem): string` — for metric: `${Math.round(valueMm * 10) / 10}mm`. For imperial: call formatFraction(valueMm / MM_PER_INCH).

5. Run tests to confirm they pass: `npx vitest run src/lib/units.test.ts --reporter=verbose`

6. Create `src/contexts/UnitContext.tsx` as a client component:
   ```typescript
   'use client';

   import { createContext, useContext, useState, type ReactNode } from 'react';
   import type { UnitSystem } from '@/lib/types';

   interface UnitContextType {
     units: UnitSystem;
     toggleUnits: () => void;
     setUnits: (units: UnitSystem) => void;
   }

   const UnitContext = createContext<UnitContextType | null>(null);

   export function UnitProvider({ children }: { children: ReactNode }) {
     const [units, setUnits] = useState<UnitSystem>('imperial');

     const toggleUnits = () => {
       setUnits(prev => prev === 'imperial' ? 'metric' : 'imperial');
     };

     return (
       <UnitContext.Provider value={{ units, toggleUnits, setUnits }}>
         {children}
       </UnitContext.Provider>
     );
   }

   export function useUnits(): UnitContextType {
     const context = useContext(UnitContext);
     if (!context) {
       throw new Error('useUnits must be used within a UnitProvider');
     }
     return context;
   }
   ```
